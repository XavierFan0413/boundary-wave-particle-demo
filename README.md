# Boundary Between Wave and Particle Demo

A small Python visualization inspired by Touhou Project spell card design, especially the idea of creating a wave-like visual effect from discrete particles.

This project explores how a pattern made entirely of discrete particles can appear wave-like when emitted with a changing angular phase.

## Features

- Interactive particle-based wave pattern visualization
- Real-time GUI controls for key parameters
- Adjustable values for:
  - ways
  - alpha
  - emit interval
  - bullet speed
  - spawn radius
- Keyboard controls for pause, reset, save snapshot, and quit

## Requirements

- Python 3
- matplotlib
- numpy

## Install

```bash
pip install -r requirements.txt
```

## Run

```bash
python boundary_wave_particle_demo.py
```

## GUI Controls

The window includes sliders for adjusting the simulation in real time:

- `ways`: number of simultaneous radial streams
- `alpha`: angular acceleration strength
- `emit`: emission interval in frames
- `speed`: bullet speed
- `spawn`: spawn ring radius

Buttons:

- `Pause`: pause or resume the animation
- `Reset`: reset the current pattern
- `Save`: save a snapshot as an image

## Keyboard Controls

- `Space`: pause or resume
- `r`: reset
- `s`: save snapshot
- `q`: quit

## What This Demonstrates

This demo is meant to show how a continuous wave-like visual effect can emerge from purely discrete particles.

The pattern is generated by emitting particles from a rotating ring, while the angular phase changes over time. Even though the system only contains individual points, the overall structure can look like a smooth wavefront.

## Notes

This is a conceptual reconstruction for learning and visualization. It is not a source-level recreation of the original Touhou implementation.
